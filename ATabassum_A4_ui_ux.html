UI/UX

I've focused on creating intuitive and engaging UI/UX designs, from responsive HUDs to clear navigation systems, to enhance player experience, improve usability, and keep players immersed in the game.

Experience

  • Designed game UI mockups for a fanstasy game usgin Figma including character selection screen.
  • Conducted user testing with 5 individuals to assess navigation, readability, and responsiveness.
  • Refined UI layout based on feedback to reduce player confusion and enhance satisfaction.
  • Designed and crafted "Neptune's Bounty", a level which helped define the tenets of BioShock level design
  • Prototyped animated menu transitions and button hover effects using Adobe XD and Figma Smart Animate for smooth in-game implementation.
  • Created accessibility-focused UI designs, including colorblind-friendly palettes and scalable font sizes to ensure inclusivity.
